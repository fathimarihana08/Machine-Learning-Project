# 🚢 Titanic Survival Prediction – Machine Learning Project

## 📌 Project Overview
This project builds a Machine Learning model to predict whether a passenger survived the Titanic disaster using historical passenger data.

The project demonstrates the full ML pipeline:
- Data preprocessing
- Feature selection
- Model training
- Model evaluation

---

## 🤖 Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest (Best Model)

---

## 🛠️ Tools & Technologies
- Python
- Jupyter Notebook
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn

---

## 📊 Model Evaluation
Models were evaluated using:
- Accuracy Score
- Confusion Matrix
- Classification Report

Random Forest achieved the best accuracy.

---

## 🧠 Conclusion
The model successfully predicts survival based on features such as gender, passenger class, and fare.
